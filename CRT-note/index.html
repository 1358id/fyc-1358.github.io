简化题意：他让我求 x≡ai(modmi)x \equiv a_i \pmod{m_i}x≡ai​(modmi​) 的解。
例如，{x≡1(mod3)x≡1(mod5)x≡2(mod7) \begin{cases} x \equiv 1 \pmod{3} \\ x \equiv 1 \pmod{5} \\ x \equiv 2 \pmod{7} \end{cases} ⎩⎪⎨⎪⎧​x≡1(mod3)x≡1(mod5)x≡2(mod7)​ 这是样例。
令 M=m1m2…mn,Mi=M/miM=m_1m_2\ldots m_n,M_i=M/m_iM=m1​m2​…mn​,Mi​=M/mi​ 。
显然 gcd⁡(Mi,mi)=1\gcd(M_i,m_i)=1gcd(Mi​,mi​)=1，所以 MiM_iMi​ 关于 mim_imi​ 的逆元存在，将其设为 tit_iti​。
于是有 Miti≡1(modmi),Miti≡0(modmj)(j≠i)M_it_i \equiv 1 \pmod{m_i},M_it_i \equiv 0 \pmod{m_j}(j\ne i)Mi​ti​≡1(modmi​),Mi​ti​≡0(modmj​)(j​=i)。
把上面每个式子左右两边同乘 aia_iai​ ，就得到 Mitiai≡ai(modmi),Mitiai≡0(modmj)(j≠i)M_it_ia_i \equiv a_i \pmod{m_i},M_it_ia_i \equiv 0 \pmod{m_j}(j\neq i)Mi​ti​ai​≡ai​(modmi​),Mi​ti​ai​≡0(modmj​)(j​=i)。

ll a[20], m[20], _M = 1, M[20], t[20];
void Exgcd(ll a, ll b, ll &x, ll &y) { 
  if (!b) x = 1, y = 0;
  else Exgcd(b, a % b, y, x), y -= a / b * x;
}
int main() {
  int n;
  ll x = 0, y = 0, ans = 0;
  read(n);
  rep(i, 1, n) read(m[i]), read(a[i]), _M *= m[i];
  rep(i, 1, n) M[i] = _M / m[i];
  rep(i, 1, n) {
    x = 0, y = 0;
    Exgcd(M[i], m[i], x, y);
    t[i] = x < 0 ? x + m[i] : x;
  }
  rep(i, 1, n) ans += (a[i] * M[i] * t[i]), ans %= _M;
  print(ans > 0 ? ans : ans + _M);
  return 0;
}
