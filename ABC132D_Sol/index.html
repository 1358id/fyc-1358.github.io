ABC132D题解

发布于 14-03-2021 | 分类于 数学 、 题解 | 2分钟 | 228字数

题目
数学题。
我要把蓝色球分割成 iii 个块，那么红球相应的被分成 i+1i+1i+1 个块。
这样就想到用隔板法。在篮球之间插 i−1i-1i−1 个板，在红球之间插 iii 个板。
但是红球的两侧也可以放蓝球，所以我们把红球个数 +2+2+2 (左右各多放一个)，蓝球放好了以后再去掉。
所以蓝球个数是 kkk 个，红球个数是 n−k+2n-k+2n−k+2 个。
容易得到答案是 (in)×(i−1k−1)\dbinom{i}{n} \times \dbinom{i-1}{k-1}(ni​)×(k−1i−1​)。用杨辉三角维护组合数即可。
代码：

ll n, k, f[2010][2010], ans;
int main() {
	read(n), read(k);
	f[0][0] = 1;
	rep(i, 1, n) {
		f[i][0] = 1;
		rep(j, 1, i) f[i][j] = (f[i - 1][j - 1] + f[i - 1][j]) % MOD;
	}
	rep(i, 1, k) print(f[n - k + 1][i] * f[k - 1][i - 1] % MOD), puts("");
	return 0;
}
