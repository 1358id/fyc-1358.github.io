错误合集

26-02-2021

  1. 未开 long long 。

  2. 线段树先调用 build()。

  3. 与上面类似，分块先调用 pre_treat() 。

  4. 分块整块 query/upd 是 rep(i, l + 1, r - 1) 。

  5. 函数 return 。

  6. 分块单点修改时 sum 数组也要相应修改。

  7. 多解的题，找到一个解以后直接 break 掉或者再去找最优解（看题目的要求）。
